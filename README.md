## Project Description

COMP 1080SEF (Group 9) project is Mathematics Educational Games, introduced by Python for primary students. With math and PyQt5 library, the project can provide a modern graphical user interface generate mathematics question (+-*/) randomly. With sqlite3, the project save wrong answers and corresponding questions to database for reviewing.
